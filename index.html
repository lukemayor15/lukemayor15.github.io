I am a recent software engineering and game development graduate that achieved a first-class honours with a letter of commendation.

Dissertation Project


Description

The project I created for my dissertation was a narrative game made in Unreal 4. The game consists of a branching dialogue system, as well as a turn-based combat system to control multiple characters.

The combat system includes abilities, basic attack, and useable items that are obtained by defeating enemies. There is also an equipment system in place that allows for characters to equip weapons and armor.

Technologies
  • Unreal 4
  • Blueprints
